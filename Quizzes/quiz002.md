    present = True

## HL Only

```.py
x = [10, 30, 10, 26]
y = [20, 15, 5,-6]
present = False
for num in range(0,4):
    a = x[num]
    b = y[num]
    the_sum = a + b
    if a == 20 or b == 20:
        present = True
    elif the_sum == 20:
        present = True
    else:
        present = False
    print(present)
```
